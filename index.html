Multi-Go

Inspired by the classical game of Go, I created a game server to play variations of the game. Traditionally, the game is played between two players on a 19x19 grid. Our game includes variations allowing multiple players on variable grid sizes. There are plans to develpo other variations such as grid configurations and point systems in the future. The project was put together using the MERN stack (MongoDB, ExpressJS, Redux and Node.js).
